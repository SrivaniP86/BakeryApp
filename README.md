# BakeryApp
🥐 BakEbUN - Digital Bakery
An enterprise-grade bakery management system built with ASP.NET Core 9 and Angular 19.

✨ Key Technical Features
* Backend: RESTful API with ASP.NET Core, EF Core, and LocalDB.

Architecture: Clean, decoupled design using DTOs and Repository patterns.

Database: SQL Server integration for robust data persistence.
